# clean-view
> 受项目 [极简财新](https://github.com/Plasma-Blue/Simple-Caixin) 启发，对一些无用元素繁杂的页面（如CSDN、财新等）进行清洁美化

## 简介
清洁网页，提升阅读体验，旨在优化网页的视觉效果，减少页面的视觉干扰；油猴脚本，支持油猴、Tampermonkey、Violentmonkey

## 已支持
* 财新周刊、详情页
* CSDN文章详情页
* 腾讯云开发者文章详情页
* Maven列表与详情页
* 机器之心文章页
* 微信文章页
* InfoQ文章页
* Reddit文章页
* Stack Overflow问题页

## 使用
油猴 - 管理界面 - 实用工具 - 导入 - 导入js文件
